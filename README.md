<h3>☑️ wolf_in_sheeps_clothing.py</h3>
<p>Warn the sheep in front of the wolf that it is about to be eaten. Remember that you are standing at the front of the queue, which is at the end of the list.
If the wolf is the closest animal to you, print "Please go away and stop eating my sheep". Otherwise, return "Oi! Sheep number N! You are about to be eaten by a wolf!" where N is the sheep's position in the queue.</p>
<div style="margin-top: 10px; margin-bottom: 20px;">
  <b>Example</b><br><br>

  <span style="color:#58a6ff;"><b>Input:</b></span>
  <pre style="background:#0d1117; color:#79c0ff; padding:10px; border-radius:6px;">sheep, sheep, sheep, wolf, sheep</pre>

  <span style="color:#7ee787;"><b>Output:</b></span>
  <pre style="background:#0d1117; color:#a5d6ff; padding:10px; border-radius:6px;">Oi! Sheep number 1! You are about to be eaten by a wolf!</pre>
</div>
<h3>☑️ sum_of_a_beach.py</h3>
<p>Beaches are filled with sand, water, fish, and sun. Given a string, calculate how many times the words "Sand", "Water", "Fish", and "Sun" appear (case insensitive).</p>
<div style="margin-top: 10px; margin-bottom: 20px;">
  <b>Example</b><br><br>

  <span style="color:#58a6ff;"><b>Input:</b></span>
  <pre style="background:#0d1117; color:#79c0ff; padding:10px; border-radius:6px;">GolDeNSanDyWateRyBeaChSuNN</pre>

  <span style="color:#7ee787;"><b>Output:</b></span>
  <pre style="background:#0d1117; color:#a5d6ff; padding:10px; border-radius:6px;">3</pre>
</div>
